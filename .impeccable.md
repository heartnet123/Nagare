## Design Context

### Users
AI engineers and developers building, evaluating, and monitoring RAG pipelines and AI agent systems. They need a functional workspace to manage complex workflows.

### Brand Personality
Minimal, Clean, Professional.

### Aesthetic Direction
High-end developer tool (inspired by platforms like Linear and Vercel). The interface should be sleek, leveraging both Light and Dark modes seamlessly. It should avoid generic AI UI patterns, gradient text, and over-the-top decorative elements in favor of precision.

### Design Principles
1. **Calm and Clarity**: The UI must evaporate cognitive load. Information hierarchy should be obvious, and spacing should provide breathing room, fostering a sense of calm.
2. **Professional Restraint**: Use subtle micro-interactions and refined typography. Avoid unnecessary glassmorphism, heavy shadows, or generic AI aesthetics.
3. **High-End Utility**: Every element must serve a purpose without feeling merely utilitarian. Emphasize crisp borders, exact alignment, and high-quality contrast.
